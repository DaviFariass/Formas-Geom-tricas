## 📖 Sobre o Projeto

Este projeto é uma aplicação web interativa projetada para facilitar o aprendizado e o cálculo rápido de propriedades geométricas. Através de uma interface amigável e visualmente atraente, o usuário pode escolher entre diversas formas geométricas e inserir as medidas necessárias para que o sistema calcule automaticamente a **Área** ou o **Perímetro**.

É um excelente projeto que demonstra a aplicação prática de lógica de programação e matemática pura utilizando JavaScript para manipulação dinâmica do DOM.

---


## ⚙️ Funcionalidades e Formas Suportadas

A aplicação possui fórmulas matemáticas integradas para calcular dinamicamente as seguintes formas:

- [x] **Círculo** (Cálculo com base no raio)
- [x] **Retângulo** e **Quadrado** (Base e altura / Lados iguais)
- [x] **Trapezoide Irregular** (Base maior, base menor e altura)
- [x] **Polígono Regular com N Lados**
- [x] **Tronco da Pirâmide**
- [x] **Coroa do Círculo** (Região anular entre dois círculos concêntricos)
- [x] **Setor Circular**

**Fluxo de Uso:**
1. O usuário seleciona a forma desejada no menu principal.
2. O sistema exibe um formulário limpo e focado, solicitando apenas as variáveis necessárias para aquela forma específica.
3. O JavaScript captura os inputs, realiza o cálculo matemático da fórmula e devolve o resultado em tempo real.
